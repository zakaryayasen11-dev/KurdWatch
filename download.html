دابەزاندن بۆ ئەندرۆید

فایلی APK · ڕاستەوخۆ دابمەزرێنە

داگرتنی APKلە ڕێگەی MediaFire
  • لینکەکە بکەرەوەلینکی سەرەوە لە وێبگەڕی مۆبایلی ئەندرۆیدەکەت بکەرەوە و کلیک لە دوگمەی Download بکە
  • فایلەکە بکەرەوەدوای تەواوبوونی داگرتن، فایلەکە بکەرەوە لە بەشی Notifications یان فۆڵدەری Downloads
  • ڕێگە بدە بە دامەزراندنئەگەر پەیامێک هات دەڵێت "ئەم جۆرە فایلە لە شوێنێکی نەزانراوەوە دێت"، کلیک لە Settings بکە و Allow from this source چالاک بکە، پاشان بگەڕێوە
  • دامەزرێنەکلیک لە Install بکە و چاوەڕوان بە
  • کردنەوە 🎉کلیک لە Open بکە — ئێستا ئامادەیت بۆ بەکارهێنان!
ئەم ئاگادارییە ئاساییە و مانای ئەوە نییە ئەپەکە کێشەی هەیە — تەنها لەبەر ئەوەی لە Google Play دانلۆد نەبووە.

دابەزاندن بۆ ویندۆز

فایلی MSIX · بۆ ویندۆز ١٠ و ١١

داگرتنی MSIXلە ڕێگەی MediaFire
  • لینکەکە بکەرەوەلینکی سەرەوە لە کۆمپیوتەری ویندۆزەکەت بکەرەوە و کلیک لە دوگمەی Download بکە
  • فایلەکە بکەرەوەدوای تەواوبوونی داگرتن، دووبارە کلیک لەسەر فایلەکە بکە (لای Downloads)
  • "Run anyway" هەڵبژێرەئەگەر پەیامێک هات دەڵێت "Windows protected your PC"، کلیک لە More info بکە پاشان Run anyway
  • دامەزرێنەکلیک لە Install بکە و چاوەڕوان بە
  • ئامادەیە 🎉ئەپەکە خۆکار دەکرێتەوە — یان بیدۆزەوە لە مینیۆی Start
ئەگەر App Installer پێشتر لەسەر سیستەمەکەت دامەزراوبێت (کە زۆرجار خۆکار هەیە لەسەر ویندۆز ١٠/١١)، دامەزراندن زۆر ئاسانترە.

دابەزاندن بۆ ئایفۆن / ئایپاد

وەک ئەپی ڕاستەقینە — بەبێ App Store

  • ماڵپەڕەکە بکەرەوە بە Safariگرنگە: تەنها لە Safari کاردەکات، نەک Chrome یان هیچ وێبگەڕێکی تر لەسەر ئایفۆن
  • دوگمەی هاوبەشکردن بدۆزەوەلە خوارەوەی شاشەکە (یان سەرەوە لەسەر iPad)، کلیک لە ئایکۆنی Share بکە
  • "Add to Home Screen" هەڵبژێرەلیستەکە خوارەوە بسڕەوە تا Add to Home Screen ببینیت، کلیکی لێ بکە
  • "Add" دابگرەناوی ئەپەکە (KURD WATCH) و ئایکۆنەکەی خۆکارانە دەردەکەوێت — تەنها کلیک لە Add لای سەرەوەی ڕاست بکە
  • ئامادەیە 🎉لۆگۆی KURD WATCH ئێستا لەسەر سەرەکی شاشەکەتە — کلیکی لێ بکە و وەک ئەپێکی ڕاستەقینە دەکرێتەوە (بەبێ نواری وێبگەڕ)
ئایفۆن ڕێگە بە دامەزراندنی ئەپی دەرەکی (لە دەرەوەی App Store) نادات، بۆیە ئەم شێوازە باشترین ڕێگەیە — هەروا کاردەکات وەک ئەپی ڕاستەقینە، ئایکۆنی خۆی و بەبێ نواری وێبگەڕ.
